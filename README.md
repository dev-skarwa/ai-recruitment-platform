# AI Recruitment Platform

Scalable backend platform for recruitment and study workflow management built using Laravel and modern backend engineering practices.

---

## Planned Features

- Authentication APIs
- Role-based access control
- Multi-tenant organizations
- Notification system
- Redis queues
- Dockerized architecture
- GitHub Actions CI/CD
- AWS deployment
- AI integrations
